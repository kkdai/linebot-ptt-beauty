## For Developer

### 本機測試

```
# 1) 請先前往 [Line Message API](https://developers.line.biz/en/services/messaging-api/) 申請 AccessKey & Secret Key 以及相關設定，在此不贅述

# 2) 啟動 Linebot + MongDB
make dev
```

### 資料注入
使用 https://github.com/mong0520/ptt-web-crawler/blob/master/run.sh，將資料注入 mongoDB 即可
